\# Smart Crowd Prediction System



A real-time crowd monitoring system that detects congestion levels using computer vision.



\## Features

\- Real-time people detection using YOLOv8

\- Crowd density classification (Low / Medium / High)

\- Live camera monitoring

\- Flask-based backend

\- Interactive dashboard



\## Technologies Used

\- Python

\- YOLOv8

\- OpenCV

\- Flask



\## Use Case

Helps monitor crowd congestion in public areas, events, and smart city environments.
